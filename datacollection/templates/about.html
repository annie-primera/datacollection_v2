About this study

What is the purpose of this research?

Through the implementation of ProWritingAid into ESL classrooms, I am trying to answer the following questions:

  • How can software technologies like AWE help to support the process of self-regulated learning in ESL students?
  • How do ESL students engage with the feedback provided by AWE programs to create their own learning?
  • Does using AWE programs help ESL students become more proficient English writers?

How will data be collected from me?

Every time you interact with the program, data is recorded automatically. The following data will be collected:

  • Time spent using the platform
  • Functions used
  • Feedback uptake
  • Submitted text

You might be randomly selected for an interview at the end of the workshop. If this happens, I will be recording the interview as audio with no images.

What will be done with the data? Will it be kept confidential?

The data will be analysed quantitatively and qualitatively. I want to look at how you use the feedback, whether it helps you engage with your texts and whether it allows you to improve the quality of your writing.

All the data will be anonymised, so it is impossible to trace back to the original student. Only a numbered ID will identify each studentâ€™s work, but the keys to those ID numbers will be stored in an encrypted file. All data collected will be stored in an encrypted form as well.

Will the results be shared?

Excerpts from the data may be shared with my PhD supervisors, but as stated above, all identifying information will be erased from them and they will be anonymised.

The results will be used as part of my doctoral thesis, and I may publish a paper based on this research. If the research is published, anonymity will be respected as stated in the previous section.

If you have any questions about this project, either now or in the future, please contact me at ana.hibert@ed.ac.uk.
